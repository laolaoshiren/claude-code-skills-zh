# 中文 README 生成器

## 触发条件
当用户要求写 README、生成项目说明、创建项目文档时激活此技能。

## README 模板结构

```markdown
# 项目名称

> 一句话描述

[![Stars](badge)](link)
[![License](badge)](link)

## ✨ 特性
- 特性 1
- 特性 2

## 🚀 快速开始

### 安装
\`\`\`bash
pip install xxx
\`\`\`

### 使用
\`\`\`python
from xxx import yyy
\`\`\`

## 📖 文档
- [完整文档](link)
- [API 参考](link)

## 🤝 贡献
欢迎 PR！

## 📝 License
MIT
```

## 工作流程
1. 分析项目结构和技术栈
2. 提取核心功能和使用方式
3. 生成中文 README
4. 包含 badge、代码示例、截图占位符
