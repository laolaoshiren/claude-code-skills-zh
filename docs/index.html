📖 什么是 Claude Code Skills？

可复用的指令模块，让 Claude Code 拥有特定领域的专业能力

⚡

即装即用

只需一个 SKILL.md 文件，复制到 ~/.claude/skills/ 目录即可生效。无需配置、无需重启。

🎯

场景驱动

每个 Skill 聚焦一个场景：代码审查、安全审计、文档生成、API 测试…让 Claude 成为领域专家。

🌏

中文友好

精选并优化适合中文开发者的技能，附带原创中文专属技能包。中英混合，技术术语保留英文。

🔄

持续更新

每周追踪社区最新技能，持续补充高质量内容。定期优化原创技能的触发条件和工作流程。

📦 安装指南

复制粘贴，30 秒搞定

⚡ 一键安装（GitHub 原生） 
# 克隆仓库
git clone https://github.com/laolaoshiren/claude-code-skills-zh.git

# 复制全部技能
cp -r claude-code-skills-zh/skills/* ~/.claude/skills/
🇨🇳 国内镜像加速（GitHub 访问困难） 
# 使用 gh-proxy 镜像克隆
git clone https://gh-proxy.com/https://github.com/laolaoshiren/claude-code-skills-zh.git

# 复制全部技能
cp -r claude-code-skills-zh/skills/* ~/.claude/skills/
📦 下载 ZIP（无需 Git） 
# GitHub 原生地址
https://github.com/laolaoshiren/claude-code-skills-zh/archive/main.zip

# 国内镜像加速
https://gh-proxy.com/https://github.com/laolaoshiren/claude-code-skills-zh/archive/main.zip

# 解压后复制技能
cp -r claude-code-skills-zh/skills/* ~/.claude/skills/
